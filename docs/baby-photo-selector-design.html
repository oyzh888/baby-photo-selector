产品概述
SECTION 01

一款帮助新手父母从海量手机照片中自动筛选出宝宝最佳照片的 iOS 工具 App。用户授权相册后，App 自动扫描最近 1000 张照片，通过本地 AI 识别宝宝照片并评分，挑出最好看的 20～50 张，支持手动微调后一键保存为系统相册中的「宝宝精选」文件夹。

核心价值主张

🔒
隐私安全
全本地处理，照片不离开手机
⚡️
一键自动筛选
无需手动翻看海量照片
✏️
结果可控
误判可手动纠正，完全掌控

Expo（Bare Workflow）：快速开发，支持 OTA 更新，后续按需迁移 SwiftUI。目标平台 iOS 优先，代码结构预留 Android 扩展空间。

AI 评分算法（V1）
SECTION 04

筛选流水线

1
人脸检测
ML Kit 检测照片中的人脸 → 无人脸直接过滤
2
年龄段判断
判断是否为婴幼儿 → 婴儿概率 < 0.5 过滤
3
综合评分
按四个维度加权计算每张照片得分
4
连拍去重
60 秒内连拍归为同组，同组保留最高分
5
输出 Top 结果
默认取 Top 30 张，同一天最多 5 张

综合评分权重（V1）

🎯 人脸清晰度35%
👶 婴儿概率30%
☀️ 曝光正常20%
📐 人脸构图15%

评分权重配置（可扩展）

// config/scoringConfig.ts
export const DEFAULT_SCORING_CONFIG = {
  weights: {
    sharpness:       0.35,  // 清晰度
    babyProbability: 0.30,  // 婴儿概率
    exposure:        0.20,  // 曝光
    composition:     0.15,  // 构图
  },
  filters: {
    minBabyProbability: 0.5,  // 低于此值直接过滤
    burstWindowSeconds: 60,   // 连拍聚类时间窗口
  },
  output: {
    defaultCount: 30,   // 默认输出张数
    maxPerDay:     5,    // 同一天最多输出张数
  },
};

🏠 首页（Home）

App Logo + 标语「帮你从海量照片里找出宝宝最美的瞬间」+ 主按钮「开始扫描」。有历史结果时额外显示上次扫描时间和「查看上次结果」入口。

🔑 授权引导页（Permission）

强调本地处理与隐私安全，提供两种授权路径：

✅ 允许访问全部照片
自动扫描最近 1000 张
📂 只选择部分照片
调起系统照片选择器，扫描选中的

⏳ 扫描进度页（Scanning）

动态进度条 + 实时文字「正在分析第 342 张…」+ 预估剩余时间。每批处理 20 张，可最小化到后台，锁屏不中断。

✨ 精选结果页（Results）— 核心页面

3 列瀑布流网格，默认全选（蓝色对勾），底部固定「保存精选」按钮。

操作行为
单张点击全屏预览（支持左右滑动切换）
单张长按弹出「移除」选项
右上角切换 Tab「AI 精选」/ 「全部照片」
「全部照片」Tab展示原始 1000 张，可点击添加到精选
顶部「重新扫描」清空结果，重新开始

🎉 完成页（Done）

📂 在系统相册查看 📤 分享给家人（系统分享面板） 🔄 重新扫描

数据模型
SECTION 06

使用 AsyncStorage 本地存储上次扫描结果：

interface ScanResult {
  scannedAt:     number;           // 时间戳
  totalScanned:  number;           // 扫描总数
  candidates:    PhotoCandidate[];  // AI 候选照片
  selected:      string[];         // 用户最终选中的 localIdentifier
}

interface PhotoCandidate {
  localIdentifier: string;   // PhotoKit 唯一 ID
  uri:             string;   // 缩略图 URI
  creationTime:    number;   // 拍摄时间
  scores: {
    sharpness:       number; // 清晰度 0~1
    babyProbability: number; // 婴儿概率 0~1
    exposure:        number; // 曝光评分 0~1
    composition:     number; // 构图评分 0~1
    total:           number; // 综合得分 0~1
  };
}

用户流程总览
SECTION 07 
启动 App
  └─> 首次打开 → 授权引导页
        ├─> 全部授权 → 扫描进度页（自动扫描最近1000张）
        └─> 部分授权 → 系统选择器 → 扫描进度页
  └─> 再次打开（有历史）→ 首页（可查看上次结果或重新扫描）

扫描完成 → 精选结果页
  ├─> 移除误判（长按 → 移除）
  ├─> 补充漏选（切换「全部照片」Tab → 点击添加）
  └─> 确认 → 「保存精选」→ 完成页

完成页
  ├─> 查看系统相册
  ├─> 分享给家人
  └─> 重新扫描
